Pass Pass c’est quoi ?

Pass Pass c’est 3 supports au choix !

Avec Pass Pass, choisissez le support de titres le plus adapté à vos besoins.
Pour accéder à tous les services et abonnements Transpole, la carte personnelle unique est faite pour vous ! La carte non-personnalisée est à adopter pour charger plusieurs types de titres. Enfin, optez pour les tickets rechargeables si vous êtes un voyageur très occasionnel.

Mais aussi, un nouveau réflexe à adopter !

Adoptez le nouveau réflexe « je bipe, je passe ».
Avec Pass Pass, quel que soit le support, ticket ou carte, que vous soyez abonné ou non abonné, c’est le même geste pour TOUS !
Que ce soit dans le bus, dans le tram, dans le métro, le TER, à chaque montée et à chaque correspondance, mon geste Pass Pass, c’est « je bipe, je passe ».

Maintien d'une double billetterie, Papier et Pass Pass

Durant la transition…
Transpole maintient ses deux systèmes de billetterie, Papier et Pass Pass, afin de permettre une transition en douceur et laisser le temps aux voyageurs de se familiariser à Pass Pass.
